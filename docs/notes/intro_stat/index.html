기초통계·회귀분석·다변량분석 강의노트

저자

권세혁

1 강의노트 소개

본 강의노트는 통계분석의 기초부터 회귀분석 및 고급 다변량분석까지 체계적으로 학습할 수 있도록 구성한 자료이다.
데이터 이해와 탐색적 분석, 통계적 추론, 예측모형 및 차원축소 기법 등을 단계적으로 학습하며 실제 데이터분석 실무에 적용할 수 있는 능력을 기르는 것을 목표로 한다.

특히 다음과 같은 학습자를 대상으로 한다.

  • 통계학 및 데이터분석 입문자
  • 사회과학·경영학 연구자
  • 머신러닝 및 AI 분석 입문자
  • 논문통계 및 실증분석 수행자
  • 공공·산업 데이터 분석 실무자

2 기초통계

통계학의 기본 개념과 데이터 분석 방법을 학습한다.

2.1 통계학의 개념

  • 기술통계와 추론통계
  • 모집단과 표본
  • 통계적 사고
  • 데이터 기반 의사결정

2.2 데이터와 통계

  • 데이터의 종류
  • 척도 수준
  • 데이터 수집 방법
  • 데이터 전처리 기초

2.3 일변량 분석

  • 평균·중앙값·최빈값
  • 분산과 표준편차
  • 왜도와 첨도
  • 히스토그램과 박스플롯

2.4 교차표 분석

  • 범주형 자료 분석
  • 교차표 작성
  • 카이제곱 검정
  • 연관성 분석

2.5 적합성 검정

  • 적합도 검정
  • 카이제곱 적합성 검정
  • 기대빈도 계산
  • 모형 적합성 평가

2.6 정규성 검정

  • 정규분포 개념
  • Shapiro-Wilk 검정
  • Kolmogorov-Smirnov 검정
  • QQ Plot 해석

2.7 상관분석

  • 공분산
  • Pearson 상관계수
  • Spearman 순위상관
  • 상관과 인과관계

2.8 분산분석(ANOVA)

  • 집단 평균 비교
  • 일원분산분석
  • 이원분산분석
  • 사후검정(Post Hoc)

2.9 시계열 분석

  • 시계열 데이터 개념
  • 추세·계절성
  • 이동평균
  • 자기상관
  • ARIMA 기초

3 회귀분석

변수 간 관계를 설명하고 예측하는 회귀모형을 학습한다.

3.1 개념·추정

  • 단순회귀분석
  • 다중회귀분석
  • 최소제곱법(OLS)
  • 회귀계수 해석
  • 결정계수(R²)

3.2 변수선택

  • Forward Selection
  • Backward Elimination
  • Stepwise 방법
  • 변수선택 기준(AIC/BIC)

3.3 다중공선성

  • 공선성의 개념
  • 분산팽창계수(VIF)
  • 조건수(Condition Number)
  • 해결 방법

3.4 회귀진단

  • 잔차분석
  • 이상치 탐지
  • 영향점 분석
  • 등분산성 검토

3.5 로지스틱회귀

  • 이항반응변수
  • 로짓(Logit) 함수
  • 오즈비(Odds Ratio)
  • 분류모형 평가

4 다변량분석

여러 변수 간 복합적인 관계를 동시에 분석하는 방법을 학습한다.

4.1 개요

  • 다변량분석의 개념
  • 고차원 데이터
  • 변수 간 구조 탐색
  • 차원축소 개요

4.2 PCA

  • 주성분분석 개념
  • 공분산행렬
  • 고유값과 고유벡터
  • 차원축소와 시각화

4.3 요인분석

  • 잠재요인 개념
  • 공통요인모형
  • 요인적재량
  • 요인회전(Varimax)

4.4 군집분석

  • 비지도학습 개념
  • 계층적 군집분석
  • K-means 군집화
  • 군집 타당성 평가

4.5 판별분석

  • 집단 분류 문제
  • 선형판별분석(LDA)
  • 판별함수
  • 분류 정확도 평가

4.6 다차원척도·대응분석

  • 거리와 유사성
  • 다차원척도법(MDS)
  • 대응분석(CA)
  • 시각적 관계 해석

4.7 정준상관·MANOVA

  • 정준상관분석
  • 다변량분산분석(MANOVA)
  • 다변량 가설검정
  • 변수 집합 간 관계 분석

5 학습 목표

본 강의노트를 통해 다음과 같은 역량을 함양하는 것을 목표로 한다.

  • 데이터 이해 및 탐색 능력 향상
  • 통계적 추론 및 검정 수행 능력 확보
  • 회귀모형 구축 및 해석 역량 강화
  • 다변량 데이터 분석 능력 함양
  • 머신러닝 및 AI 학습을 위한 통계 기반 확보

본 자료는 다음과 같은 분야에서 활용할 수 있다.

  • 경영분석
  • 사회조사분석
  • 정책평가
  • 금융데이터분석
  • 의료·바이오 통계

7 마무리

현대 데이터분석에서는 단순 통계처리를 넘어 복합적 데이터 구조를 이해하고 예측하는 능력이 중요하다.
본 강의노트가 통계학적 사고력과 데이터분석 역량을 체계적으로 향상시키는 데 도움이 되기를 바란다.
