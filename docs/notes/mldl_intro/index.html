머신러닝·딥러닝 강의노트

저자

권세혁

1 강의노트 소개

본 강의노트는 머신러닝(Machine Learning)과 딥러닝(Deep Learning)의 핵심 개념부터 실제 예측·분류 문제 적용까지 체계적으로 학습할 수 있도록 구성한 자료이다.

전통적 통계모형과 현대 머신러닝 알고리즘을 함께 비교하며 학습하고, 차원축소·군집화·예측·분류·딥러닝 사례분석까지 단계적으로 다룬다.

특히 다음과 같은 학습자를 대상으로 한다.

  • 머신러닝 및 AI 입문자
  • 데이터사이언스 전공 학생
  • 통계 기반 AI 연구자
  • 실무 데이터분석가
  • 딥러닝 프로젝트 수행자

2 머신·딥러닝 개념

머신러닝과 딥러닝의 기본 원리와 학습체계를 이해한다.

2.1 개요

  • 인공지능·머신러닝·딥러닝 관계
  • 데이터 기반 학습
  • 알고리즘 중심 분석
  • AI 활용 분야

2.2 MLDL 개념 1

  • 모델(Model)의 개념
  • 학습(Training)
  • 추론(Inference)
  • 일반화(Generalization)

2.3 MLDL 개념 2

  • 과적합(Overfitting)
  • 편향-분산 트레이드오프
  • 하이퍼파라미터
  • 교차검증(Cross Validation)

2.4 지도학습

  • 입력과 출력
  • 회귀와 분류
  • 학습 데이터 구조
  • 예측모형 구축

2.5 비지도학습

  • 레이블 없는 데이터
  • 군집화
  • 차원축소
  • 패턴 탐색

2.6 MLDL 평가

  • 정확도(Accuracy)
  • 정밀도·재현율
  • RMSE·MAE
  • ROC-AUC

2.7 불확실성

  • 확률적 예측
  • 예측구간
  • 베이지안 관점
  • 불확실성 정량화

2.8 MLDL 기초

  • 데이터 전처리
  • Feature Engineering
  • 학습 파이프라인
  • 모델 선택

2.9 딥러닝 기초

  • 인공신경망(ANN)
  • 활성화함수
  • 역전파(Backpropagation)
  • Optimizer와 Epoch
