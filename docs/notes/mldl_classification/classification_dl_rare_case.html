1. лҚ°мқҙн„°

лҚ°мқҙн„° л¶Ҳлҹ¬мҳӨкё°

Credit Card Fraud(creditcard.csv) лҚ°мқҙн„°лҠ” лҢҖн‘ңм Ғмқё мқҙм§„ л¶„лҘҳ(surveillance / fraud detection) лІӨм№ҳл§ҲнҒ¬лЎң, н•ң кұҙмқҳ кұ°лһҳк°Җ мӮ¬кё°(Fraud) мқём§Җ м •мғҒ(Normal) мқём§ҖлҘј нҢҗм •н•ҳлҠ” л¬ём ңлҘј лӢӨлЈ¬лӢӨ.

л°ҳмқ‘ліҖмҲҳлҠ” Class н•ҳлӮҳмқҙл©° ліҙнҶө Class=0мқ„ м •мғҒ(negative), Class=1мқ„ мӮ¬кё°(positive)лЎң м •мқҳн•ңлӢӨ. мҰү лӘ©н‘ңлҠ” к°Ғ кұ°лһҳ \(P(Y=1вҲЈX=x)\) мқ„ м¶”м •н•ҳкі , мҡҙмҳҒ лӘ©м Ғ(мҳӨнғҗ/лҜёнғҗ л№„мҡ©, кІҖнҶ лҹү л“ұ)м—җ л§һлҠ” мһ„кі„к°’ tлҘј м •н•ҙ \(\hat y =1{p(x)вүҘt}\) нҳ•нғңлЎң мқҳмӮ¬кІ°м •мқ„ лӮҙлҰ¬лҠ” кІғмқҙлӢӨ.

лҚ°мқҙн„° к·ңлӘЁлҠ” 284,807кұҙмқҳ кұ°лһҳлЎң кө¬м„ұлҗҳл©°, мҙқ 31к°ң ліҖмҲҳлҘј к°Җм§„лӢӨ. мқҙ мӨ‘ Classк°Җ нғҖк№ғ(y)мқҙкі  лӮҳлЁём§Җ 30к°ңк°Җ мһ…л Ҙ ліҖмҲҳ(X) лӢӨ. мһ…л Ҙ ліҖмҲҳлҠ” Time, V1~V28, AmountлЎң мқҙлЈЁм–ҙм§„лӢӨ. мһҗлЈҢнҳ• кҙҖм җм—җм„ң мһ…л Ҙ ліҖмҲҳл“ӨмқҖ лҢҖл¶Җл¶„ м—°мҶҚнҳ•мқҙл©°(мӢӨмҲҳнҳ•), Classл§Ң м •мҲҳнҳ• лқјлІЁмқҙлӢӨ.

нҠ№нһҲ V1~V28мқҖ мӣҗлһҳ кұ°лһҳ мҶҚм„ұл“Өмқ„ к·ёлҢҖлЎң м ңкіөн•ң кІғмқҙ м•„лӢҲлқј, к°ңмқём •ліҙ ліҙнҳё л°Ҹ л№„мӢқлі„нҷ”лҘј мң„н•ҙ PCA(мЈјм„ұл¶„ л¶„м„қ) лЎң ліҖнҷҳлҗң нҠ№м§•л“ӨмқҙлӢӨ. л”°лқјм„ң к°Ғ \(V_k\) лҠ” мӣҗліҖмҲҳл“Өмқҳ м„ нҳ•кІ°н•©мңјлЎң м–»м–ҙм§„ мЈјм„ұл¶„ м җмҲҳмқҙл©°, мӣҗлһҳ ліҖмҲҳмқҳ н•ҙм„қ к°ҖлҠҘм„ұмқҖ м ңн•ңлҗңлӢӨ.

л°ҳл©ҙ Timeкіј AmountлҠ” л№„көҗм Ғ м§ҒкҙҖм Ғмқё мқҳлҜёлҘј мң м§Җн•ңлӢӨ. TimeмқҖ кё°мӨҖ мӢңм җ мқҙнӣ„мқҳ кІҪкіј мӢңк°„мңјлЎң мӢңк°„лҢҖм—җ л”°лҘё нҢЁн„ҙмқ„ л°ҳмҳҒн•  мҲҳ мһҲкі , AmountлҠ” кұ°лһҳ кёҲм•ЎмңјлЎңм„ң мӮ¬кё° м—¬л¶ҖмҷҖ м—°кҙҖлҗң мӨ‘мҡ”н•ң мӢ нҳёк°Җ лҗ  мҲҳ мһҲлӢӨ.

мқҙ лҚ°мқҙн„°мқҳ к°ҖмһҘ кІ°м •м Ғмқё нҠ№м„ұмқҖ к·№мӢ¬н•ң нҒҙлһҳмҠӨ л¶Ҳк· нҳ•(extreme class imbalance) мқҙлӢӨ. м •мғҒ кұ°лһҳ(Class=0)лҠ” 284,315кұҙмңјлЎң м „мІҙмқҳ м•Ҫ 99.827%лҘј м°Ём§Җн•ҳлҠ” л°ҳл©ҙ, мӮ¬кё° кұ°лһҳ(Class=1)лҠ” 492кұҙмңјлЎң м•Ҫ 0.173%м—җ л¶Ҳкіјн•ҳлӢӨ.

мқҙлҹ° нҷҳкІҪм—җм„ңлҠ” лӢЁмҲң м •нҷ•лҸ„(Accuracy)к°Җ мүҪкІҢ кіјлҢҖнҸүк°ҖлҗңлӢӨ. мҳҲлҘј л“Өм–ҙ лӘЁл“  кұ°лһҳлҘј м •мғҒмңјлЎңл§Ң мҳҲмёЎн•ҙлҸ„ м •нҷ•лҸ„лҠ” м•Ҫ 99.8%м—җ лӢ¬н•  мҲҳ мһҲмңјлӮҳ, мқҙлҠ” мӮ¬кё° нғҗм§ҖлқјлҠ” лӘ©м Ғмқ„ м „нҳҖ лӢ¬м„ұн•ҳм§Җ лӘ»н•ңлӢӨ. л”°лқјм„ң мӢӨл¬ҙм ҒмңјлЎңлҠ” м–‘м„ұ(мӮ¬кё°) нғҗм§Җ м„ұлҠҘмқ„ м§Ғм ‘ л°ҳмҳҒн•ҳлҠ” м§Җн‘ң, мҳҲм»ЁлҢҖ PrecisionвҖ“Recall(PR) кҙҖм җмқҳ нҸүк°Җ(AP нҸ¬н•Ё)лӮҳ, мҳӨнғҗмқ„ мқјм • мҲҳмӨҖ мқҙн•ҳлЎң м ңн•ңн•ҳлҠ” FPR м ңм•Ҫ кё°л°ҳ мһ„кі„к°’ м„ нғқ, нҳ№мқҖ лҜёнғҗ(FN)кіј мҳӨнғҗ(FP)мқҳ л№„мҡ©мқ„ л°ҳмҳҒн•ң л№„мҡ©кё°л°ҳ мһ„кі„к°’ м„Өкі„к°Җ н•өмӢ¬мқҙ лҗңлӢӨ.

мқҙ мҪ”л“ңлҠ” нқ¬к·Җ мқҙлІӨнҠё(мӮ¬кё° кұ°лһҳ) мқҙм§„ л¶„лҘҳ л¬ём ңм—җм„ң н•ҷмҠөкіј нҸүк°Җк°Җ мҷңкіЎлҗҳм§Җ м•ҠлҸ„лЎқ лҚ°мқҙн„° л¶„лҰ¬мҷҖ м „мІҳлҰ¬лҘј лҲ„мҲҳ м—Ҷмқҙ мқјкҙҖлҗҳкІҢ мҲҳн–үн•ҳкё° мң„н•ң кө¬м„ұмқҙлӢӨ.

лЁјм Җ train_test_splitмңјлЎң м „мІҙ лҚ°мқҙн„°лҘј н•ҷмҠөмҡ©(train)кіј н…ҢмҠӨнҠёмҡ©(test)мңјлЎң л¶„лҰ¬н•ҳлҠ” лӢЁкі„мқҙлӢӨ. мқҙл•Ң stratify=yлҘј мӮ¬мҡ©н•ҳм—¬ мӮ¬кё°(Class=1) л№„мңЁмқҙ trainкіј testм—җ кұ°мқҳ лҸҷмқјн•ҳкІҢ мң м§ҖлҗҳлҸ„лЎқ н•ҳлҠ” л°©мӢқмқҙлӢӨ. мӮ¬кё° лҚ°мқҙн„°лҠ” м „мІҙм—җм„ң к·№нһҲ м ҒмқҖ л¶Ҳк· нҳ• лҚ°мқҙн„°мқҙлҜҖлЎң stratifyлҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡмңјл©ҙ н…ҢмҠӨнҠём…Ӣмқҳ м–‘м„ұ л№„мңЁмқҙ мҡ°м—°м—җ мқҳн•ҙ нқ”л“ӨлҰ¬кі , к·ё кІ°кіј нҸүк°Җ м§Җн‘ңк°Җ л¶Ҳм•Ҳм •н•ҙм§Ҳ к°ҖлҠҘм„ұмқҙ нҒ° кө¬мЎ°мқҙлӢӨ. м¶ңл Ҙлҗң Fraud rateк°Җ trainкіј testм—җм„ң кұ°мқҳ к°ҷкІҢ лӮҳнғҖлӮҳлҠ” кІғмқҖ л¶„лҰ¬ кіјм •мқҙ мқҳлҸ„лҢҖлЎң мҲҳн–үлҗҳм—ҲмқҢмқ„ ліҙм—¬мЈјлҠ” кІ°кіјмқҙлӢӨ.

лӢӨмқҢмқҖ ColumnTransformerлҘј мқҙмҡ©н•ң м „мІҳлҰ¬ м •мқҳ лӢЁкі„мқҙлӢӨ. мқҙ лҚ°мқҙн„°мқҳ V1~V28мқҖ PCA кё°л°ҳмңјлЎң ліҖнҷҳлҗң мҲҳм№ҳ нҠ№м„ұмқҙлҜҖлЎң лҢҖмІҙлЎң мҠӨмјҖмқјмқҙ мң мӮ¬н•ң нҳ•нғңлЎң м ңкіөлҗҳлҠ” нҺёмқҙлӢӨ. л°ҳл©ҙ AmountмҷҖ TimeмқҖ мӣҗлһҳ мҠӨмјҖмқјмқ„ мң м§Җн•ҳлҠ” ліҖмҲҳмқҙлҜҖлЎң к°’мқҳ лІ”мң„к°Җ мғҒлҢҖм ҒмңјлЎң нҒ¬кі  н•ҷмҠө м•Ҳм •м„ұмқ„ м Җн•ҙн•  мҲҳ мһҲлҠ” мҡ”мҶҢмқҙлӢӨ. л”°лқјм„ң cols_scaleмқ„ [вҖңAmountвҖқ, вҖңTimeвҖқ]мңјлЎң м§Җм •н•ҳкі , н•ҙлӢ№ л‘җ м»¬лҹјм—җл§Ң StandardScalerлҘј м Ғмҡ©н•ҳм—¬ нҸүк·  0, н‘ңмӨҖнҺём°Ё 1лЎң н‘ңмӨҖнҷ”н•ҳлҠ” кө¬м„ұмқҙлӢӨ. лӮҳлЁём§Җ м»¬лҹјмқҖ remainder=вҖңpassthroughвҖқлЎң м§Җм •н•ҳм—¬ ліҖнҷҳ м—Ҷмқҙ к·ёлҢҖлЎң нҶөкіјмӢңнӮӨлҠ” л°©мӢқмқҙлӢӨ. мқҙлҠ” н•„мҡ”н•ң м»¬лҹјм—җл§Ң мөңмҶҢ м „мІҳлҰ¬лҘј м Ғмҡ©н•ҳлҠ” мӢӨл¬ҙнҳ• м„ нғқмқҙлӢӨ.

м „мІҳлҰ¬м—җм„ң к°ҖмһҘ мӨ‘мҡ”н•ң мӣҗм№ҷмқҖ лҚ°мқҙн„° лҲ„мҲҳ л°©м§Җ мӣҗм№ҷмқҙлӢӨ. м „мІҳлҰ¬ к°қмІҙ preprocessлҠ” н•ҷмҠө лҚ°мқҙн„°м—җм„ңл§Ң fitмқ„ мҲҳн–үн•ҙм•ј н•ҳл©°, н…ҢмҠӨнҠё лҚ°мқҙн„°м—җлҠ” transformл§Ң м Ғмҡ©лҗҳм–ҙм•ј н•ңлӢӨ. fit_transform(X_train)мқҖ trainм—җм„ң нҸүк· кіј н‘ңмӨҖнҺём°Ё к°ҷмқҖ нҶөкі„лҹүмқ„ н•ҷмҠөн•ҳкі  лҸҷмӢңм—җ ліҖнҷҳн•ҳлҠ” кіјм •мқҙлӢӨ. transform(X_test)мқҖ trainм—җм„ң н•ҷмҠөн•ң кё°мӨҖмқ„ лҸҷмқјн•ҳкІҢ testм—җ м Ғмҡ©н•ҳлҠ” кіјм •мқҙлӢӨ. л§Ңм•Ҫ н…ҢмҠӨнҠё лҚ°мқҙн„°к№Ңм§Җ нҸ¬н•Ён•ҳм—¬ мҠӨмјҖмқјлҹ¬лҘј fitн•ҙлІ„лҰ¬л©ҙ н…ҢмҠӨнҠё м •ліҙк°Җ м „мІҳлҰ¬ кіјм •м—җ л°ҳмҳҒлҗҳлҠ” лҲ„мҲҳ л¬ём ңк°Җ л°ңмғқн•ҳкі , к·ё кІ°кіј м„ұлҠҘмқҙ кіјлҢҖнҸүк°ҖлҗҳлҠ” мң„н—ҳмқҙ мЎҙмһ¬н•ҳлҠ” кө¬мЎ°мқҙлӢӨ.

set_output(transform=вҖңpandasвҖқ)лҘј мӮ¬мҡ©н•  мҲҳ мһҲлҠ” нҷҳкІҪм—җм„ңлҠ” м „мІҳлҰ¬ кІ°кіјлҘј pandas DataFrameмңјлЎң л°ӣм•„ м»¬лҹјлӘ…мқ„ ліҙмЎҙн•ҳлҠ” кө¬м„ұмқҙлӢӨ. мқҙл•Ң feature_namesлҠ” ліҖнҷҳ кІ°кіј DataFrameмқҳ columnsлҘј нҶөн•ҙ к·ёлҢҖлЎң нҷ•ліҙлҗҳлҠ” л°©мӢқмқҙлӢӨ. мқҙнӣ„ л”Ҙлҹ¬лӢқ лӘЁлҚё мһ…л Ҙмқ„ мң„н•ҙ to_numpy()лЎң л„ҳнҢҢмқҙ л°°м—ҙлЎң ліҖнҷҳн•ҳм—¬ X_train_scaledмҷҖ X_test_scaledлҘј мғқм„ұн•ҳлҠ” нқҗлҰ„мқҙлӢӨ. л§Ңм•Ҫ set_outputмқҙ м§Җмӣҗлҗҳм§Җ м•ҠлҠ” нҷҳкІҪмқҙл©ҙ мҳҲмҷё мІҳлҰ¬лЎң л„ҳм–ҙк°Җл©°, мқҙ кІҪмҡ° ColumnTransformerмқҳ м¶ңл ҘмқҖ ліҙнҶө л°°м—ҙ нҳ•нғңмқҙлҜҖлЎң feature_namesлҠ” м§Ғм ‘ кө¬м„ұн•ҳкІҢ лҗңлӢӨ. мқҙл•Ң ColumnTransformer нҠ№м„ұмғҒ ліҖнҷҳлҗң м»¬лҹј(Amount, Time)мқҙ м•һмӘҪм—җ л°°м№ҳлҗҳкі , remainderлЎң нҶөкіјлҗң лӮҳлЁём§Җ м»¬лҹјмқҙ л’Өм—җ мқҙм–ҙм§ҖлҠ” мҲңм„ңк°Җ лҗңлӢӨ. мӢӨм ң м¶ңл Ҙм—җм„ң feature_namesмқҳ м•һл¶Җл¶„мқҙ [вҖҳAmountвҖҷ, вҖҳTimeвҖҷ, вҖҳV1вҖҷ, вҖҳV2вҖҷ, вҖҳV3вҖҷ] нҳ•нғңлЎң лӮҳнғҖлӮҳлҠ” кІғмқҖ мқҙлҹ¬н•ң м¶ңл Ҙ мҲңм„ңлҘј л°ҳмҳҒн•ң кІ°кіјмқҙлӢӨ.

мў…н•©н•ҳл©ҙ, мқҙ мҪ”л“ңлҠ” л¶Ҳк· нҳ• лҚ°мқҙн„°м—җм„ң л¶„нҸ¬лҘј м•Ҳм •м ҒмңјлЎң мң м§Җн•ң train/test л¶„лҰ¬лҘј мҲҳн–үн•ҳкі , AmountмҷҖ Timeм—җл§Ң н‘ңмӨҖнҷ”лҘј м Ғмҡ©н•ҳл©°, м „мІҳлҰ¬мқҳ fitмқ„ trainм—җл§Ң кі м •н•ЁмңјлЎңмҚЁ лҲ„мҲҳлҘј л°©м§Җн•ҳкі , л”Ҙлҹ¬лӢқ н•ҷмҠөмқ„ мң„н•ң л°°м—ҙкіј нҠ№м„ұ мқҙлҰ„к№Ңм§Җ мқјкҙҖлҗҳкІҢ мӨҖл№„н•ҳлҠ” м „мІҳлҰ¬ нҢҢмқҙн”„лқјмқё кө¬м„ұмқҙлӢӨ.

2. кіөнҶө мӨҖл№„ м…Җ (TensorFlow + seed + к°ҖмӨ‘м№ҳ)

мқҙ мҪ”л“ңлҠ” н…җм„ңн”ҢлЎң кё°л°ҳ л”Ҙлҹ¬лӢқ н•ҷмҠөмқ„ мӢңмһ‘н•ҳкё° м „м—җ мһ¬нҳ„м„ұмқ„ нҷ•ліҙн•ҳкі , к·№лӢЁм Ғ нҒҙлһҳмҠӨ л¶Ҳк· нҳ• л¬ём ңлҘј мҷ„нҷ”н•ҳкё° мң„н•ң н‘ңліё к°ҖмӨ‘м№ҳ(sample weight)лҘј кі„мӮ°н•ҳлҠ” мӨҖл№„ лӢЁкі„мқҙлӢӨ.

лЁјм Җ numpyмҷҖ tensorflow, keras л°Ҹ layersлҘј мһ„нҸ¬нҠён•ҳлҠ” л¶Җл¶„мқҙлӢӨ. мқҙнӣ„ SEED=42лҘј м •н•ҳкі  np.random.seed(SEED), tf.random.set_seed(SEED)лҘј нҳём¶ңн•ҳлҠ”лҚ°, мқҙлҠ” лӮңмҲҳм—җ мқҳмЎҙн•ҳлҠ” м—°мӮ°(к°ҖмӨ‘м№ҳ мҙҲкё°нҷ”, лҜёлӢҲл°°м№ҳ м…”н”Ң, л“ңлЎӯм•„мӣғ л“ұ)мқҳ кІ°кіјк°Җ мӢӨн–үл§ҲлӢӨ лӢ¬лқјм§ҖлҠ” нҳ„мғҒмқ„ мӨ„м—¬ мӢӨн—ҳ мһ¬нҳ„м„ұмқ„ лҶ’мқҙкё° мң„н•ң м„Өм •мқҙлӢӨ. мҷ„м „н•ң мһ¬нҳ„м„ұмқҖ мӢӨн–ү нҷҳкІҪмқҙлӮҳ м—°мӮ° м»Өл„җм—җ л”°лқј м ңн•ңлҗ  мҲҳ мһҲмңјлӮҳ, мқјл°ҳм ҒмңјлЎң мқҙ л‘җ мӨ„мқҖ мӢӨн—ҳ л№„көҗм—җ н•„мҡ”н•ң мөңмҶҢн•ңмқҳ кі м • мһҘм№ҳмқҙлӢӨ.

лӢӨмқҢмқҖ нҒҙлһҳмҠӨ л¶Ҳк· нҳ•мқ„ ліҙм •н•ҳкё° мң„н•ң к°ҖмӨ‘м№ҳ кі„мӮ° лӢЁкі„мқҙлӢӨ. n_pos = int(y_train.sum())лҠ” н•ҷмҠө лҚ°мқҙн„°м—җм„ң м–‘м„ұ нҒҙлһҳмҠӨ(мӮ¬кё°, y=1)мқҳ к°ңмҲҳлҘј мқҳлҜён•ңлӢӨ. лқјлІЁмқҙ 0кіј 1лЎң кө¬м„ұлҗҳм–ҙ мһҲмңјлҜҖлЎң н•©(sum)мқҖ 1мқҳ к°ңмҲҳмҷҖ лҸҷмқјн•ң кө¬мЎ°мқҙлӢӨ. n_neg = int(len(y_train) - n_pos)лҠ” м „мІҙ н•ҷмҠө н‘ңліё мҲҳм—җм„ң м–‘м„ұ к°ңмҲҳлҘј ләҖ к°’мқҙл©°, мқҢм„ұ нҒҙлһҳмҠӨ(м •мғҒ, y=0)мқҳ к°ңмҲҳмқҙлӢӨ.

л§Ҳм§Җл§ү print л¬ёмқҖ n_pos, n_neg, pos_weightлҘј м¶ңл Ҙн•ҳм—¬ к°ҖмӨ‘м№ҳк°Җ м–ҙлҠҗ м •лҸ„лЎң м„Өм •лҗҳм—ҲлҠ”м§Җ нҷ•мқён•ҳкё° мң„н•ң м§„лӢЁмҡ© м¶ңл ҘмқҙлӢӨ. мқҙнӣ„ лӘЁлҚё н•ҷмҠөм—җм„ң model.fit(вҖҰ, sample_weight=w_train) нҳ•нғңлЎң w_trainмқ„ м „лӢ¬н•ҳл©ҙ, мқҙ к°ҖмӨ‘м№ҳк°Җ л°°м№ҳ лӢЁмң„ мҶҗмӢӨ кі„мӮ°м—җ л°ҳмҳҒлҗҳлҠ” кө¬мЎ°мқҙлӢӨ.

(1) MLP + к°ҖмӨ‘ BCE (к¶ҢмһҘ л©”мқё)

мқҙ мҪ”л“ңлҠ” к·№лӢЁм Ғ л¶Ҳк· нҳ•(мӮ¬кё° л№„мңЁмқҙ л§Өмҡ° лӮ®мқҖ) мқҙм§„ л¶„лҘҳ л¬ём ңм—җм„ң, лӢӨмёөнҚјм…үнҠёлЎ (MLP)мқ„ кө¬м„ұн•ҳкі  к°ҖмӨ‘ мҶҗмӢӨмқ„ м Ғмҡ©н•ҳм—¬ н•ҷмҠөн•ң л’Ө н…ҢмҠӨнҠё лҚ°мқҙн„°м—җ лҢҖн•ң мҳҲмёЎ нҷ•лҘ мқ„ мғқм„ұн•ҳлҠ” нқҗлҰ„мқҙлӢӨ.

лЁјм Җ build_mlp(input_dim) н•ЁмҲҳлҠ” мһ…л Ҙ нҠ№м„ұ мҲҳ(input_dim)лҘј л°ӣм•„ MLP лӘЁлҚёмқ„ м •мқҳн•ҳлҠ” н•ЁмҲҳмқҙлӢӨ. мһ…л ҘмёөмқҖ keras.Input(shape=(input_dim,))лЎң м„Өм •лҗҳм–ҙ мһҲмңјл©°, к°Ғ мғҳн”Ңмқҙ кёёмқҙ input_dimмқё 1м°Ёмӣҗ лІЎн„°мһ„мқ„ мқҳлҜён•ңлӢӨ. мқҙнӣ„ Dense(64, activation=вҖңreluвҖқ) мёөмқҙ мІ« лІҲм§ё мқҖлӢүмёөмңјлЎң л°°м№ҳлҗҳм–ҙ 64м°Ёмӣҗ н‘ңнҳ„мқ„ н•ҷмҠөн•ҳлҸ„лЎқ кө¬м„ұлҗҳм–ҙ мһҲлӢӨ. к·ё лӢӨмқҢ Dropout(0.2)мқҙ м Ғмҡ©лҗҳлҠ”лҚ°, н•ҷмҠө мӢң мқҖлӢү мң лӢӣмқҳ мқјл¶ҖлҘј нҷ•лҘ м ҒмңјлЎң л№„нҷңм„ұнҷ”н•ҳм—¬ кіјм Ғн•©мқ„ мҷ„нҷ”н•ҳлҠ” м •к·ңнҷ” кё°лІ•мқҙлӢӨ. л‘җ лІҲм§ё мқҖлӢүмёөмқҖ Dense(32, activation=вҖңreluвҖқ)лЎң 32м°Ёмӣҗ н‘ңнҳ„мқ„ м¶”к°ҖлЎң н•ҷмҠөн•ҳлҸ„лЎқ кө¬м„ұлҗҳм–ҙ мһҲмңјл©°, лӢӨмӢң Dropout(0.2)мңјлЎң м •к·ңнҷ”лҘј н•ң лІҲ лҚ” м Ғмҡ©н•ңлӢӨ. м¶ңл ҘмёөмқҖ Dense(1, activation=вҖңsigmoidвҖқ)лЎң м •мқҳлҗҳл©°, мӢңк·ёлӘЁмқҙл“ң нҷңм„ұнҷ”мқҳ м¶ңл ҘмқҖ 0кіј 1 мӮ¬мқҙмқҳ к°’мқҙлҜҖлЎң к°Ғ кұ°лһҳк°Җ мӮ¬кё°мқј нҷ•лҘ лЎң н•ҙм„қлҗҳлҠ” м җмҲҳ(score)лҘј мғқм„ұн•ҳлҠ” кө¬мЎ°мқҙлӢӨ. л§Ҳм§Җл§үмңјлЎң keras.Model(inputs, outputs)лҘј л°ҳнҷҳн•ҳм—¬ мһ…л Ҙм—җм„ң м¶ңл Ҙк№Ңм§Җмқҳ кі„мӮ° к·ёлһҳн”„лҘј н•ҳлӮҳмқҳ лӘЁлҚё к°қмІҙлЎң л§Ңл“ лӢӨ.

mlp_bce = build_mlp(X_train_scaled.shape[1])лҠ” м „мІҳлҰ¬лҗң н•ҷмҠө лҚ°мқҙн„°мқҳ м—ҙ к°ңмҲҳ(нҠ№м„ұ мҲҳ)лҘј мһ…л Ҙ м°ЁмӣҗмңјлЎң мӮ¬мҡ©н•ҳм—¬ лӘЁлҚёмқ„ мӢӨм ңлЎң мғқм„ұн•ҳлҠ” лӢЁкі„мқҙлӢӨ. мқҙм–ҙм§ҖлҠ” compile лӢЁкі„лҠ” н•ҷмҠөмқ„ мң„н•ң мөңм Ғнҷ” м„Өм •кіј мҶҗмӢӨн•ЁмҲҳ л°Ҹ нҸүк°Җ м§Җн‘ңлҘј м§Җм •н•ҳлҠ” лӢЁкі„мқҙлӢӨ. optimizerлҠ” Adam(1e-3)лЎң м§Җм •лҗҳм–ҙ мһҲмңјл©°, мқҙлҠ” н•ҷмҠөлҘ  0.001мқҳ Adam мөңм Ғнҷ” м•Ңкі лҰ¬мҰҳмңјлЎң к°ҖмӨ‘м№ҳлҘј к°ұмӢ н•ңлӢӨлҠ” мқҳлҜёмқҙлӢӨ. lossлҠ” BinaryCrossentropyлЎң м§Җм •лҗҳм–ҙ мқҙм§„ л¶„лҘҳм—җ н‘ңмӨҖм ҒмңјлЎң мӮ¬мҡ©н•ҳлҠ” лЎңк·ё мҶҗмӢӨмқ„ мөңмҶҢнҷ”н•ҳлҸ„лЎқ н•ҷмҠөлҗңлӢӨ. metricsм—җлҠ” AUC(curve=вҖңROCвҖқ)мҷҖ AUC(curve=вҖңPRвҖқ)к°Җ нҸ¬н•Ёлҗҳм–ҙ мһҲлҠ”лҚ°, л¶Ҳк· нҳ• лҚ°мқҙн„°м—җм„ңлҠ” ROC-AUCлҝҗ м•„лӢҲлқј PR-AUCк°Җ лӘЁлҚёмқҳ м–‘м„ұ(мӮ¬кё°) л¶„лі„л Ҙмқ„ лҚ” м§Ғм ‘м ҒмңјлЎң л°ҳмҳҒн•ҳлҠ” кІҪмҡ°к°Җ л§Һкё° л•Ңл¬ём—җ л‘җ м§Җн‘ңлҘј н•Ёк»ҳ лӘЁлӢҲн„°л§Ғн•ҳлҠ” кө¬м„ұмқҙлӢӨ.

cbм—җ нҸ¬н•Ёлҗң EarlyStopping мҪңл°ұмқҖ кІҖмҰқ м§Җн‘ңк°Җ к°ңм„ лҗҳм§Җ м•Ҡмқ„ л•Ң н•ҷмҠөмқ„ мЎ°кё°м—җ мў…лЈҢн•ҳкё° мң„н•ң мһҘм№ҳмқҙлӢӨ. monitor=вҖңval_pr_aucвҖқмҷҖ mode=вҖңmaxвҖқлҠ” кІҖмҰқ PR-AUCк°Җ мөңлҢҖк°Җ лҗҳлҸ„лЎқ н•ҷмҠөмқ„ м§„н–үн•ҳл©°, мқҙ к°’мқҙ лҚ” мқҙмғҒ к°ңм„ лҗҳм§Җ м•Ҡмңјл©ҙ л©Ҳм¶”лҸ„лЎқ м„Өм •н•ң кІғмқҙлӢӨ. patience=3мқҖ м—°мҶҚ 3 epoch лҸҷм•Ҳ к°ңм„ мқҙ м—Ҷмқ„ л•Ң мў…лЈҢн•ңлӢӨлҠ” мқҳлҜёмқҙл©°, restore_best_weights=TrueлҠ” н•ҷмҠөмқҙ мў…лЈҢлҗң мӢңм җмқҳ к°ҖмӨ‘м№ҳк°Җ м•„лӢҲлқј кІҖмҰқ PR-AUCк°Җ к°ҖмһҘ лҶ’м•ҳлҚҳ мӢңм җмқҳ к°ҖмӨ‘м№ҳлҘј ліөмӣҗн•ҳлҸ„лЎқ н•ҳлҠ” м„Өм •мқҙлӢӨ. мқҙ м„Өм •мқҖ л¶Ҳн•„мҡ”н•ң кіјм Ғн•© кө¬к°„м—җ м§„мһ…н•ң л’Өмқҳ к°ҖмӨ‘м№ҳлҘј лӮЁкё°м§Җ м•Ҡкё° мң„н•ң лӘ©м ҒмқҙлӢӨ.

hist_bce = mlp_bce.fit(вҖҰ)лҠ” мӢӨм ң н•ҷмҠө мҲҳн–ү лӢЁкі„мқҙлӢӨ. мһ…л ҘмқҖ X_train_scaledмҷҖ y_trainмқҙл©°, sample_weight=w_trainмқҙ н•өмӢ¬ м„Өм •мқҙлӢӨ. w_trainмқҖ м–‘м„ұ мғҳн”Ңмқҳ мҶҗмӢӨ кё°м—¬лҸ„лҘј нҒ¬кІҢ л§Ңл“Өм–ҙ л¶Ҳк· нҳ•мңјлЎң мқён•ҙ лӘЁлҚёмқҙ м •мғҒ(0)л§Ң мҳҲмёЎн•ҳлҠ” л°©н–ҘмңјлЎң мҲҳл ҙн•ҳлҠ” нҳ„мғҒмқ„ мҷ„нҷ”н•ҳлҠ” лӘ©м ҒмқҙлӢӨ. validation_split=0.2лҠ” н•ҷмҠө лҚ°мқҙн„°мқҳ мқјл¶Җ(20%)лҘј лӮҙл¶Җ кІҖмҰқмҡ©мңјлЎң л–јм–ҙ кІҖмҰқ м§Җн‘ңлҘј кі„мӮ°н•ҳлҠ” л°©мӢқмқҙлӢӨ. epochs=20мқҖ мөңлҢҖ 20лІҲмқҳ epochмқ„ мҲҳн–үн•ҳлҗҳ, EarlyStopping мЎ°кұҙмқҙ л§ҢмЎұлҗҳл©ҙ лҚ” мқјм°Қ мў…лЈҢлҗҳлҠ” кө¬мЎ°мқҙлӢӨ. batch_size=2048мқҖ н•ң лІҲмқҳ к°ҖмӨ‘м№ҳ м—…лҚ°мқҙнҠём—җ мӮ¬мҡ©н•ҳлҠ” лҜёлӢҲл°°м№ҳ нҒ¬кё°мқҙл©°, лҚ°мқҙн„°к°Җ нҒ° кІҪмҡ° н•ҷмҠө мҶҚлҸ„мҷҖ м•Ҳм •м„ұмқ„ кі л Өн•ң л№„көҗм Ғ нҒ° л°°м№ҳ м„Өм •мқҙлӢӨ.

н•ҷмҠөмқҙ лҒқлӮң л’Ө p_test_bce = mlp_bce.predict(X_test_scaled, batch_size=8192).ravel()лҠ” н…ҢмҠӨнҠё лҚ°мқҙн„°м—җ лҢҖн•ң мҳҲмёЎмқ„ мҲҳн–үн•ҳлҠ” лӢЁкі„мқҙлӢӨ. predictмқҳ м¶ңл ҘмқҖ к°Ғ мғҳн”Ңм—җ лҢҖн•ң мӢңк·ёлӘЁмқҙл“ң нҷ•лҘ мқҙлҜҖлЎң мӮ¬кё°мқј к°ҖлҠҘм„ұ м җмҲҳлЎң мӮ¬мҡ©лҗңлӢӨ. batch_size=8192лҠ” мҳҲмёЎ мӢң кі„мӮ° нҡЁмңЁмқ„ мң„н•ҙ лҚ” нҒ° л°°м№ҳлҘј мӮ¬мҡ©н•ң м„Өм •мқҙлӢӨ. ravel()мқҖ (N, 1) нҳ•нғңмқҳ м¶ңл Ҙмқ„ (N,) нҳ•нғңмқҳ 1м°Ёмӣҗ л°°м—ҙлЎң нҺҙм„ң мқҙнӣ„ нҸүк°Җ мҪ”л“ңм—җм„ң лӢӨлЈЁкё° мүҪкІҢ л§Ңл“ңлҠ” мІҳлҰ¬мқҙлӢӨ. л§Ҳм§Җл§ү printлҠ” мҳҲмёЎ нҷ•лҘ мқҳ м•һл¶Җл¶„ лӘҮ к°ңлҘј м¶ңл Ҙн•ҳм—¬ к°’мқҳ лІ”мң„мҷҖ нҳ•нғңк°Җ м •мғҒмқём§Җ нҷ•мқён•ҳлҠ” м җкІҖ лӢЁкі„мқҙлӢӨ.

м •лҰ¬н•ҳл©ҙ мқҙ мҪ”л“ңлҠ” MLP кө¬мЎ°лҘј м •мқҳн•ҳкі , мқҙм§„ көҗм°Ём—”нҠёлЎңн”ј мҶҗмӢӨкіј AUC м§Җн‘ңлҘј кё°мӨҖмңјлЎң н•ҷмҠөн•ҳл©°, л¶Ҳк· нҳ•мқ„ sample_weightлЎң ліҙм •н•ҳкі , кІҖмҰқ PR-AUCлҘј кё°мӨҖмңјлЎң мЎ°кё° мў…лЈҢн•ң л’Ө н…ҢмҠӨнҠём…Ӣ нҷ•лҘ  м җмҲҳлҘј мғқм„ұн•ҳлҠ” м „нҳ•м Ғмқё л¶Ҳк· нҳ• мқҙм§„ л¶„лҘҳ н•ҷмҠө нҢҢмқҙн”„лқјмқёмқҙлӢӨ.

def build_mlp(input_dim):
    inputs = keras.Input(shape=(input_dim,))
    x = layers.Dense(64, activation="relu")(inputs)
    x = layers.Dropout(0.2)(x)
    x = layers.Dense(32, activation="relu")(x)
    x = layers.Dropout(0.2)(x)
    outputs = layers.Dense(1, activation="sigmoid")(x)
    return keras.Model(inputs, outputs)

mlp_bce = build_mlp(X_train_scaled.shape[1])

    optimizer=keras.optimizers.Adam(1e-3),
    loss=keras.losses.BinaryCrossentropy(),
    metrics=[keras.metrics.AUC(curve="ROC", name="roc_auc"),
             keras.metrics.AUC(curve="PR",  name="pr_auc")]
)

cb = [
    keras.callbacks.EarlyStopping(monitor="val_pr_auc", mode="max",
                                  patience=3, restore_best_weights=True)
]

hist_bce = mlp_bce.fit(
    X_train_scaled, y_train,
    sample_weight=w_train,
    validation_split=0.2,
    epochs=20,
    batch_size=2048,
    callbacks=cb,
    verbose=1
)

p_test_bce = mlp_bce.predict(X_test_scaled, batch_size=8192).ravel()

(2) MLP + Focal Loss (л¶Ҳк· нҳ•м—җм„ң recall л°Җ л•Ң мһҗмЈј мң лҰ¬)

мқҙ мҪ”л“ңлҠ” л¶Ҳк· нҳ• мқҙм§„ л¶„лҘҳм—җм„ң мһҗмЈј мӮ¬мҡ©н•ҳлҠ” focal lossлҘј м§Ғм ‘ м •мқҳн•ҳкі , лҸҷмқјн•ң MLP кө¬мЎ°м—җ focal lossлҘј м Ғмҡ©н•ҳм—¬ н•ҷмҠөн•ң л’Ө н…ҢмҠӨнҠё лҚ°мқҙн„°м—җ лҢҖн•ң мҳҲмёЎ нҷ•лҘ мқ„ мғқм„ұн•ҳлҠ” кіјм •мқҙлӢӨ.

лЁјм Җ binary_focal_loss(gamma=2.0, alpha=0.25)лҠ” focal loss н•ЁмҲҳлҘј л°ҳнҷҳн•ҳлҠ” нҢ©нҶ лҰ¬ н•ЁмҲҳмқҙлӢӨ. gammaмҷҖ alphaлҠ” focal lossмқҳ л‘җ н•ҳмқҙнҚјнҢҢлқјлҜён„°мқҙл©°, gammaлҠ” мү¬мҡҙ мғҳн”Ңмқҳ кё°м—¬лҸ„лҘј мӨ„мқҙлҠ” м •лҸ„лҘј кІ°м •н•ҳкі  alphaлҠ” нҒҙлһҳмҠӨ л¶Ҳк· нҳ•мқ„ мҷ„нҷ”н•ҳкё° мң„н•ң нҒҙлһҳмҠӨ к°ҖмӨ‘ м—ӯн• мқ„ н•ңлӢӨ. лӮҙл¶Җмқҳ loss(y_true, y_pred)лҠ” мјҖлқјмҠӨк°Җ л°°м№ҳ лӢЁмң„лЎң нҳём¶ңн•ҳлҠ” мӢӨм ң мҶҗмӢӨ кі„мӮ° н•ЁмҲҳмқҙлӢӨ.

loss н•ЁмҲҳм—җм„ң y_trueлҘј float32лЎң ліҖнҷҳн•ҳлҠ” мқҙмң лҠ” мқҙнӣ„ м—°мӮ°мқ„ н…җм„ңн”ҢлЎң нғҖмһ…мңјлЎң м•Ҳм •м ҒмңјлЎң мІҳлҰ¬н•ҳкё° мң„н•ЁмқҙлӢӨ. eps = tf.keras.backend.epsilon()лҘј мӮ¬мҡ©н•ҳкі  y_predлҘј [eps, 1-eps] лІ”мң„лЎң clipн•ҳлҠ” мқҙмң лҠ” log(pt) кі„мӮ°м—җм„ң 0 лҳҗлҠ” 1мқҙ л“Өм–ҙк°Җл©ҙ лЎңк·ёк°Җ л°ңмӮ°н•ҳкұ°лӮҳ мҲҳм№ҳ л¬ём ңк°Җ мғқкё°лҠ” кІғмқ„ л°©м§Җн•ҳкё° мң„н•ң мІҳлҰ¬мқҙлӢӨ.

к·ё лӢӨмқҢ ptлҘј м •мқҳн•ҳлҠ” л¶Җл¶„мқҖ focal lossмқҳ н•өмӢ¬мқҙлӢӨ. ptлҠ” м •лӢө нҒҙлһҳмҠӨм—җ лҢҖн•ҙ лӘЁлҚёмқҙ л¶Җм—¬н•ң нҷ•лҘ мқ„ мқҳлҜён•ңлӢӨ. y_trueк°Җ 1мқҙл©ҙ ptлҠ” y_predк°Җ лҗҳкі , y_trueк°Җ 0мқҙл©ҙ ptлҠ” 1 - y_predк°Җ лҗңлӢӨ. мҰү, лӘЁлҚёмқҙ м •лӢөмқ„ л§һнһҗмҲҳлЎқ ptк°Җ м»Өм§Җкі , нӢҖлҰҙмҲҳлЎқ ptк°Җ мһ‘м•„м§ҖлҠ” кө¬мЎ°мқҙлӢӨ. wлҠ” alphaлҘј мқҙмҡ©н•ң нҒҙлһҳмҠӨ к°ҖмӨ‘м№ҳмқҙлӢӨ. y_trueк°Җ 1мқҙл©ҙ alphaлҘј, y_trueк°Җ 0мқҙл©ҙ 1-alphaлҘј мӮ¬мҡ©н•ҳм—¬ м–‘м„ұкіј мқҢм„ұмқҳ мҶҗмӢӨ л№„мӨ‘мқ„ мЎ°м Ҳн•ҳлҠ” кө¬м„ұмқҙлӢӨ.

л°ҳнҷҳмӢқмқё -mean(w * (1-pt)^gamma * log(pt))к°Җ focal lossмқҳ нҳ•нғңмқҙлӢӨ. log(pt)лҠ” мқҙм§„ көҗм°Ём—”нҠёлЎңн”јмқҳ кё°ліё нҳ•нғңмқҙл©°, (1-pt)^gammaк°Җ м¶”к°Җлҗҳм–ҙ мһҲлӢӨ. мқҙ н•ӯмқҖ ptк°Җ нҒ° мү¬мҡҙ мғҳн”Ңм—җм„ңлҠ” (1-pt)к°Җ мһ‘м•„м ё мҶҗмӢӨ кё°м—¬к°Җ кёүкІ©нһҲ мӨ„м–ҙл“Өкі , ptк°Җ мһ‘мқҖ м–ҙл Өмҡҙ мғҳн”Ңм—җм„ңлҠ” (1-pt)к°Җ м»Өм ё мҶҗмӢӨ кё°м—¬к°Җ мғҒлҢҖм ҒмңјлЎң мң м§ҖлҗҳлҸ„лЎқ л§Ңл“ңлҠ” м—ӯн• мқҙлӢӨ. кІ°кіјм ҒмңјлЎң лӘЁлҚёмқҙ лӢӨмҲҳмқҳ мү¬мҡҙ м •мғҒ кұ°лһҳм—җ кіјлҸ„н•ҳкІҢ лҒҢл Өк°Җм§Җ м•Ҡкі , мҶҢмҲҳмқҙл©ҙм„ң м–ҙл Өмҡҙ мӮ¬кё° кұ°лһҳлҘј лҚ” м§‘мӨ‘м ҒмңјлЎң н•ҷмҠөн•ҳлҸ„лЎқ мң лҸ„н•ҳлҠ” мҶҗмӢӨ н•ЁмҲҳмқҙлӢӨ.

мқҙнӣ„ mlp_focal = build_mlp(вҖҰ)лҠ” м•һм—җм„ң м •мқҳн•ң лҸҷмқјн•ң MLP кө¬мЎ°лҘј мғқм„ұн•ҳлҠ” л¶Җл¶„мқҙлӢӨ. compile лӢЁкі„м—җм„ң optimizerлҠ” Adam(1e-3)лЎң м„Өм •лҗҳм–ҙ мһҲмңјл©°, lossм—җ binary_focal_loss(gamma=2.0, alpha=0.25)лҘј м§Җм •н•ҳм—¬ н•ҷмҠө лӘ©н‘ңлҘј focal loss мөңмҶҢнҷ”лЎң л°”кҫёлҠ” кө¬м„ұмқҙлӢӨ. metricsм—җлҠ” roc_aucмҷҖ pr_aucлҘј н•Ёк»ҳ л‘җм–ҙ ROC кҙҖм җкіј PR кҙҖм җмқ„ лҸҷмӢңм—җ лӘЁлӢҲн„°л§Ғн•ҳлҸ„лЎқ н•ң м„Өм •мқҙлӢӨ. нҠ№нһҲ л¶Ҳк· нҳ• л¬ём ңм—җм„ңлҠ” PR-AUCк°Җ м„ұлҠҘ ліҖнҷ”лҘј лҚ” лҜјк°җн•ҳкІҢ л°ҳмҳҒн•ҳлҠ” кІҪмҡ°к°Җ л§ҺлӢӨ.

мҪңл°ұ cbлҠ” EarlyStoppingмңјлЎң кө¬м„ұлҗҳм–ҙ мһҲмңјл©°, кІҖмҰқ PR-AUC(val_pr_auc)лҘј кё°мӨҖмңјлЎң м„ұлҠҘмқҙ лҚ” мқҙмғҒ мўӢм•„м§Җм§Җ м•Ҡмңјл©ҙ н•ҷмҠөмқ„ мӨ‘лӢЁн•ҳлҠ” л°©мӢқмқҙлӢӨ. patience=3мқҖ 3 epoch лҸҷм•Ҳ к°ңм„ мқҙ м—Ҷмңјл©ҙ мӨ‘лӢЁн•ңлӢӨлҠ” мқҳлҜёмқҙл©°, restore_best_weights=TrueлҠ” кІҖмҰқ PR-AUCк°Җ мөңкі мҳҖлҚҳ мӢңм җмқҳ к°ҖмӨ‘м№ҳлҘј ліөмӣҗн•ҳлҠ” м„Өм •мқҙлӢӨ. мқҙлҠ” кіјм Ғн•© кө¬к°„м—җ л“Өм–ҙк°„ л’Өмқҳ к°ҖмӨ‘м№ҳлҘј лӮЁкё°м§Җ м•Ҡкё° мң„н•ң лӘ©м ҒмқҙлӢӨ.

hist_focal = mlp_focal.fit(вҖҰ)лҠ” мӢӨм ң н•ҷмҠө лӢЁкі„мқҙлӢӨ. validation_split=0.2лЎң н•ҷмҠө лҚ°мқҙн„° мқјл¶ҖлҘј кІҖмҰқм—җ мӮ¬мҡ©н•ҳл©°, epochs=20мқҖ мөңлҢҖ л°ҳліө нҡҹмҲҳмқҙлӢӨ. batch_size=2048мқҖ лҜёлӢҲл°°м№ҳ нҒ¬кё°мқҙлӢӨ. м•һм„  к°ҖмӨ‘ BCE лІ„м „кіј лӢ¬лҰ¬ м—¬кё°м„ңлҠ” sample_weightлҘј л”°лЎң мЈјм§Җ м•ҠлҠ”лҚ°, focal loss мһҗмІҙк°Җ alphaмҷҖ (1-pt)^gamma н•ӯмқ„ нҶөн•ҙ л¶Ҳк· нҳ•кіј мү¬мҡҙ мғҳн”Ң м§Җл°° л¬ём ңлҘј мҷ„нҷ”н•ҳлҸ„лЎқ м„Өкі„лҗң мҶҗмӢӨмқҙкё° л•Ңл¬ёмқҙлӢӨ. лӢӨл§Ң н•„мҡ”н•ҳл©ҙ focal lossмҷҖ sample_weightлҘј н•Ёк»ҳ м“°лҠ” кө¬м„ұлҸ„ к°ҖлҠҘн•ҳлӮҳ, ліҙнҶөмқҖ н•ң к°Җм§Җ м „лһөмқ„ мӨ‘мӢ¬мңјлЎң л№„көҗн•ҳлҠ” нҺёмқҙ к№”лҒ”н•ҳлӢӨ.

л§Ҳм§Җл§үмңјлЎң p_test_focal = mlp_focal.predict(вҖҰ).ravel()лҠ” н…ҢмҠӨнҠё лҚ°мқҙн„°м—җ лҢҖн•ң мҳҲмёЎмқ„ мҲҳн–үн•ҳлҠ” лӢЁкі„мқҙлӢӨ. м¶ңл ҘмқҖ мӢңк·ёлӘЁмқҙл“ң нҷ•лҘ мқҙлҜҖлЎң к°Ғ кұ°лһҳк°Җ мӮ¬кё°мқј к°ҖлҠҘм„ұ м җмҲҳлЎң н•ҙм„қлҗңлӢӨ. ravel()мқҖ (N,1) нҳ•нғңлҘј (N,) нҳ•нғңлЎң нҺјміҗ мқҙнӣ„ нҸүк°Җ мҪ”л“ңм—җм„ң лӢӨлЈЁкё° мүҪкІҢ л§Ңл“ңлҠ” мІҳлҰ¬мқҙлӢӨ. print л¬ёмқҖ мҳҲмёЎ к°’мқҙ м •мғҒм ҒмңјлЎң мғқм„ұлҗҳм—ҲлҠ”м§Җ м•һл¶Җл¶„ лӘҮ к°ңлҘј нҷ•мқён•ҳлҠ” м җкІҖ лӢЁкі„мқҙлӢӨ.

def binary_focal_loss(gamma=2.0, alpha=0.25):
    # y_true, y_pred in [0,1]
    def loss(y_true, y_pred):
        y_true = tf.cast(y_true, tf.float32)
        eps = tf.keras.backend.epsilon()
        y_pred = tf.clip_by_value(y_pred, eps, 1.0 - eps)

        pt = tf.where(tf.equal(y_true, 1.0), y_pred, 1.0 - y_pred)
        w  = tf.where(tf.equal(y_true, 1.0), alpha, 1.0 - alpha)

        return -tf.reduce_mean(w * tf.pow(1.0 - pt, gamma) * tf.math.log(pt))
    return loss

mlp_focal = build_mlp(X_train_scaled.shape[1])
mlp_focal.compile(
    optimizer=keras.optimizers.Adam(1e-3),
    loss=binary_focal_loss(gamma=2.0, alpha=0.25),
    metrics=[keras.metrics.AUC(curve="ROC", name="roc_auc"),
             keras.metrics.AUC(curve="PR",  name="pr_auc")]
)

cb = [
    keras.callbacks.EarlyStopping(monitor="val_pr_auc", mode="max",
                                  patience=3, restore_best_weights=True)
]

hist_focal = mlp_focal.fit(
    X_train_scaled, y_train,
    validation_split=0.2,
    epochs=20,
    batch_size=2048,
    callbacks=cb,
    verbose=1
)

p_test_focal = mlp_focal.predict(X_test_scaled, batch_size=8192).ravel()
print("p_test_focal:", p_test_focal[:5])
(3) мҳӨнҶ мқёмҪ”лҚ”(Anomaly Detection) вҖ” м •мғҒл§ҢмңјлЎң н•ҷмҠө нӣ„ мһ¬кө¬м„ұмҳӨм°ЁлЎң м җмҲҳнҷ”

мқҙ мҪ”л“ңлҠ” мҳӨнҶ мқёмҪ”лҚ”(autoencoder)лҘј мқҙмҡ©н•ң мқҙмғҒнғҗм§Җ(anomaly detection) л°©мӢқмңјлЎң, м •мғҒ кұ°лһҳл§Ңмқ„ мӮ¬мҡ©н•ҙ мһ¬кө¬м„ұ лӘЁлҚёмқ„ н•ҷмҠөн•ҳкі  н…ҢмҠӨнҠё лҚ°мқҙн„°мқҳ мһ¬кө¬м„ұ мҳӨм°ЁлҘј мқҙмғҒ м җмҲҳлЎң кі„мӮ°н•ҳлҠ” м Ҳм°ЁмқҙлӢӨ.

мІ« мӨ„м—җм„ң X_train_norm = X_train_scaled[y_train == 0]мқ„ нҶөн•ҙ н•ҷмҠө лҚ°мқҙн„° мӨ‘ м •мғҒ нҒҙлһҳмҠӨ(0)л§Ң м¶”м¶ңн•ҳлҠ” лӢЁкі„мқҙлӢӨ. мқҙлҠ” мҳӨнҶ мқёмҪ”лҚ”к°Җ м •мғҒ нҢЁн„ҙмқ„ мһҳ мһ¬кө¬м„ұн•ҳлҸ„лЎқ н•ҷмҠөмӢңнӮӨкі , м •мғҒкіј лӢӨлҘё нҢЁн„ҙ(мӮ¬кё°)мқҙ л“Өм–ҙмҳӨл©ҙ мһ¬кө¬м„ұ мҳӨм°Ёк°Җ м»Өм§ҖлҸ„лЎқ л§Ңл“ңлҠ” м „нҳ•м Ғмқё м •мғҒкё°л°ҳ(one-class) н•ҷмҠө м „лһөмқҙлӢӨ. мқҙм–ҙм§ҖлҠ” м¶ңл ҘмқҖ м •мғҒ н•ҷмҠө мғҳн”Ң мҲҳмҷҖ мһ…л Ҙ м°Ёмӣҗ(нҠ№м„ұ мҲҳ)мқҙ мқҳлҸ„лҢҖлЎң кө¬м„ұлҗҳм—ҲлҠ”м§Җ нҷ•мқён•ҳлҠ” м җкІҖмҡ© м¶ңл ҘмқҙлӢӨ.

build_autoencoder(input_dim) н•ЁмҲҳлҠ” мһ…л Ҙ м°Ёмӣҗ input_dimмқ„ л°ӣм•„ мҳӨнҶ мқёмҪ”лҚ” лӘЁлҚёмқ„ м •мқҳн•ҳлҠ” л¶Җл¶„мқҙлӢӨ. мһ…л ҘмёөмқҖ keras.Input(shape=(input_dim,))лЎң м„Өм •лҗҳм–ҙ мһҲмңјл©°, к°Ғ мғҳн”Ңмқҙ кёёмқҙ input_dimмқё 1м°Ёмӣҗ лІЎн„°мһ„мқ„ мқҳлҜён•ңлӢӨ. мқҙнӣ„ Dense(32)мҷҖ Dense(16) мёөмқҖ мқёмҪ”лҚ” м—ӯн• мқ„ н•ҳл©° мһ…л Ҙмқ„ м җм°Ё м Җм°Ёмӣҗ н‘ңнҳ„мңјлЎң м••м¶•н•ҳлҠ” кө¬мЎ°мқҙлӢӨ. z = Dense(8) мёөмқҖ лі‘лӘ©(bottleneck) лҳҗлҠ” мһ мһ¬лІЎн„°(latent representation)м—җ н•ҙлӢ№н•ҳл©°, м •мғҒ лҚ°мқҙн„°мқҳ н•өмӢ¬ кө¬мЎ°лҘј 8м°ЁмӣҗмңјлЎң мҡ”м•Ҫн•ң н‘ңнҳ„мқ„ н•ҷмҠөн•ҳлҠ” лӢЁкі„мқҙлӢӨ. к·ё лӢӨмқҢ Dense(16), Dense(32) мёөмқҖ л””мҪ”лҚ” м—ӯн• мқ„ н•ҳл©° мһ мһ¬лІЎн„°м—җм„ң мӣҗлһҳ м°ЁмӣҗмңјлЎң лӢӨмӢң ліөмӣҗн•ҳлҠ” кіјм •мқҙлӢӨ. л§Ҳм§Җл§ү м¶ңл Ҙмёө Dense(input_dim, activation=None)мқҖ мһ…л Ҙкіј лҸҷмқјн•ң м°Ёмӣҗмқҳ мһ¬кө¬м„ұ лІЎн„°лҘј мғқм„ұн•ҳлҠ” мёөмқҙл©°, activation=NoneмқҖ м„ нҳ• м¶ңл ҘмқҙлҜҖлЎң мһ…л Ҙк°’мқҳ м—°мҶҚм Ғмқё мһ¬кө¬м„ұм—җ м Ғн•©н•ң м„Өм •мқҙлӢӨ.

ae = build_autoencoder(X_train_scaled.shape[1])лҠ” нҠ№м„ұ мҲҳлҘј мһ…л Ҙ м°ЁмӣҗмңјлЎң мӮ¬мҡ©н•ҳм—¬ мҳӨнҶ мқёмҪ”лҚ”лҘј мғқм„ұн•ҳлҠ” лӢЁкі„мқҙлӢӨ. compileм—җм„ңлҠ” optimizerлҘј Adam(1e-3)лЎң м§Җм •н•ҳкі  lossлҘј вҖңmseвҖқлЎң м§Җм •н•ңлӢӨ. м—¬кё°м„ң mseлҠ” мһ…л Ҙкіј м¶ңл Ҙ(мһ¬кө¬м„ұ)мқҳ нҸүк· м ңкіұмҳӨм°ЁлҘј мөңмҶҢнҷ”н•ҳлҸ„лЎқ н•ҷмҠөн•ңлӢӨлҠ” мқҳлҜёмқҙл©°, мҳӨнҶ мқёмҪ”лҚ” н•ҷмҠөм—җ к°ҖмһҘ кё°ліём ҒмңјлЎң мӮ¬мҡ©лҗҳлҠ” мҶҗмӢӨ н•ЁмҲҳмқҙлӢӨ.

EarlyStopping мҪңл°ұмқҖ кІҖмҰқ мҶҗмӢӨ(val_loss)мқ„ кё°мӨҖмңјлЎң н•ҷмҠөмқ„ мЎ°кё°м—җ мў…лЈҢн•ҳлҠ” мһҘм№ҳмқҙлӢӨ. monitor=вҖңval_lossвҖқ, mode=вҖңminвҖқмқҖ кІҖмҰқ MSEк°Җ мһ‘м•„м§ҖлҠ” л°©н–ҘмңјлЎң н•ҷмҠөмқ„ м§„н–үн•ҳлҗҳ к°ңм„ мқҙ л©Ҳм¶”л©ҙ мӨ‘лӢЁн•ңлӢӨлҠ” мқҳлҜёмқҙлӢӨ. patience=3мқҖ 3 epoch м—°мҶҚ к°ңм„ мқҙ м—Ҷмқ„ л•Ң мў…лЈҢн•ңлӢӨлҠ” м„Өм •мқҙл©°, restore_best_weights=TrueлҠ” кІҖмҰқ мҶҗмӢӨмқҙ к°ҖмһҘ лӮ®м•ҳлҚҳ мӢңм җмқҳ к°ҖмӨ‘м№ҳлҘј ліөмӣҗн•ҳм—¬ кіјм Ғн•© кө¬к°„мқҳ к°ҖмӨ‘м№ҳк°Җ лӮЁм§Җ м•ҠлҸ„лЎқ н•ҳлҠ” м„Өм •мқҙлӢӨ.

hist_ae = ae.fit(X_train_norm, X_train_norm, вҖҰ)лҠ” н•ҷмҠө мҲҳн–ү лӢЁкі„мқҙлӢӨ. мҳӨнҶ мқёмҪ”лҚ”лҠ” мһ…л Ҙмқ„ мһҗкё° мһҗмӢ мңјлЎң мһ¬кө¬м„ұн•ҳлҠ” лӘЁлҚёмқҙлҜҖлЎң мһ…л Ҙкіј нғҖк№ғмқҙ лҸҷмқјн•ҳкІҢ X_train_normмңјлЎң л“Өм–ҙк°„лӢӨ. validation_split=0.2лҠ” м •мғҒ н•ҷмҠө лҚ°мқҙн„° мӨ‘ мқјл¶ҖлҘј кІҖмҰқмҡ©мңјлЎң л¶„лҰ¬н•ҙ мЎ°кё° мў…лЈҢ кё°мӨҖмқ„ кі„мӮ°н•ҳкё° мң„н•ң м„Өм •мқҙлӢӨ. epochs=30мқҖ мөңлҢҖ 30лІҲ л°ҳліө н•ҷмҠөмқ„ н—Ҳмҡ©н•ҳлҗҳ мЎ°кё° мў…лЈҢ мЎ°кұҙмқҙ л§ҢмЎұлҗҳл©ҙ лҚ” мқјм°Қ лҒқлӮҳлҠ” кө¬мЎ°мқҙлӢӨ. batch_size=2048мқҖ н•ң лІҲмқҳ м—…лҚ°мқҙнҠём—җ мӮ¬мҡ©н•ҳлҠ” л°°м№ҳ нҒ¬кё°мқҙл©°, лҚ°мқҙн„°к°Җ нҒ° мғҒнҷ©м—җм„ң нҡЁмңЁмқ„ кі л Өн•ң нҒ° л°°м№ҳ м„Өм •мқҙлӢӨ.

н•ҷмҠө нӣ„ Xhat_test = ae.predict(X_test_scaled, вҖҰ)лҠ” н…ҢмҠӨнҠё лҚ°мқҙн„°лҘј мҳӨнҶ мқёмҪ”лҚ”лЎң мһ¬кө¬м„ұн•ң кІ°кіјлҘј м–»лҠ” лӢЁкі„мқҙлӢӨ. мқҙм–ҙм„ң score_test_ae = np.mean((X_test_scaled - Xhat_test)**2, axis=1)мқҖ к°Ғ мғҳн”Ңлі„ мһ¬кө¬м„ұ мҳӨм°ЁлҘј кі„мӮ°н•ҳлҠ” л¶Җл¶„мқҙлӢӨ. (X - Xhat)мқҳ м ңкіұмҳӨм°ЁлҘј нҠ№м„ұ м¶•(axis=1)мңјлЎң нҸүк· лӮҙлҜҖлЎң мғҳн”Ңл§ҲлӢӨ н•ҳлӮҳмқҳ мҠӨм№јлқј м җмҲҳк°Җ мғқм„ұлҗңлӢӨ. мқҙ м җмҲҳлҠ” нҒҙмҲҳлЎқ мһ…л Ҙмқ„ м •мғҒ нҢЁн„ҙмңјлЎң мһ¬кө¬м„ұн•ҳкё° м–ҙл Өмӣ лӢӨлҠ” мқҳлҜёмқҙлҜҖлЎң мқҙмғҒ(anomaly) к°ҖлҠҘм„ұмқҙ лҶ’лӢӨкі  н•ҙм„қн•ҳлҠ” м җмҲҳмқҙлӢӨ. л§Ҳм§Җл§ү м¶ңл ҘмқҖ кі„мӮ°лҗң мқҙмғҒ м җмҲҳк°Җ м •мғҒм ҒмңјлЎң мғқм„ұлҗҳм—ҲлҠ”м§Җ нҷ•мқён•ҳкё° мң„н•ң м җкІҖмҡ© м¶ңл ҘмқҙлӢӨ.

м •лҰ¬н•ҳл©ҙ мқҙ мҪ”л“ңлҠ” м •мғҒ лҚ°мқҙн„°лЎңл§Ң мҳӨнҶ мқёмҪ”лҚ”лҘј н•ҷмҠөмӢңнӮӨкі , н…ҢмҠӨнҠё мғҳн”Ңмқҳ мһ¬кө¬м„ұ мҳӨм°ЁлҘј мқҙмғҒ м җмҲҳлЎң ліҖнҷҳн•ҳм—¬ мқҙнӣ„ лӢЁкі„м—җм„ң мһ„кі„к°’ кё°мӨҖ(Top-K, л№„мҡ© кё°л°ҳ, м •л°ҖлҸ„ м ңм•Ҫ л“ұ)мңјлЎң мӮ¬кё° мқҳмӢ¬ кұ°лһҳлҘј м„ лі„н•  мҲҳ мһҲлҸ„лЎқ мӨҖл№„н•ҳлҠ” мҪ”л“ңмқҙлӢӨ.

# м •мғҒ лҚ°мқҙн„°л§ҢмңјлЎң н•ҷмҠө
X_train_norm = X_train_scaled[y_train == 0]
print("AE train normals:", X_train_norm.shape)

def build_autoencoder(input_dim):
    inputs = keras.Input(shape=(input_dim,))
    x = layers.Dense(32, activation="relu")(inputs)
    x = layers.Dense(16, activation="relu")(x)
    z = layers.Dense(8, activation="relu")(x)
    x = layers.Dense(16, activation="relu")(z)
    x = layers.Dense(32, activation="relu")(x)
    outputs = layers.Dense(input_dim, activation=None)(x)
    return keras.Model(inputs, outputs)

ae = build_autoencoder(X_train_scaled.shape[1])
ae.compile(optimizer=keras.optimizers.Adam(1e-3), loss="mse")

cb = [
    keras.callbacks.EarlyStopping(monitor="val_loss", mode="min",
                                  patience=3, restore_best_weights=True)
]

hist_ae = ae.fit(
    X_train_norm, X_train_norm,
    validation_split=0.2,
    epochs=30,
    batch_size=2048,
    callbacks=cb,
    verbose=1
)

# мһ¬кө¬м„ұ мҳӨм°ЁлҘј anomaly scoreлЎң мӮ¬мҡ©

4. нҸүк°Җ м§Җн‘ң вҖңм„ё л°©лІ• кіөнҶө л№„көҗн‘ңвҖқ мҪ”л“ң

мқҙ мҪ”л“ңлҠ” м„ё к°Җм§Җ лӘЁлҚёмқҙ л§Ңл“  м җмҲҳ(score)лҘј лҸҷмқјн•ң кё°мӨҖмңјлЎң л№„көҗн•ҳкё° мң„н•ҙ кіөнҶө нҸүк°Җ м§Җн‘ңлҘј кі„мӮ°н•ҳкі , кІ°кіјлҘј н‘ң нҳ•нғңмқҳ DataFrameмңјлЎң м •лҰ¬н•ҳлҠ” нҸүк°Җ м…ҖмқҙлӢӨ. м—¬кё°м„ң scoreлҠ” л¶„лҘҳлӘЁлҚёмқҳ кІҪмҡ° мӮ¬кё° нҷ•лҘ  pмқҙл©°, мҳӨнҶ мқёмҪ”лҚ”мқҳ кІҪмҡ° мһ¬кө¬м„ұмҳӨм°ЁлЎңм„ң к°’мқҙ нҒҙмҲҳлЎқ мӮ¬кё° мқҳмӢ¬мқҙ нҒ° м җмҲҳмқҙлӢӨ.

лЁјм Җ roc_auc_score, average_precision_score, precision_recall_curveлҘј л¶Ҳлҹ¬мҷҖ ROC-AUC, PR-AUC(AP), к·ёлҰ¬кі  м •л°ҖлҸ„-мһ¬нҳ„мңЁ кіЎм„ мқ„ кё°л°ҳмңјлЎң н•ң мһ„кі„к°’ кҙҖл Ё кі„мӮ°мқ„ мҲҳн–үн•  мӨҖл№„лҘј н•ңлӢӨ. мқҙнӣ„ precision_recall_at_k н•ЁмҲҳлҠ” мҡҙмҳҒм—җм„ң мһҗмЈј м“°лҠ” мғҒмң„ Kкұҙ м„ лі„ м •мұ…мқ„ лӘЁмӮ¬н•ҳкё° мң„н•ң н•ЁмҲҳмқҙлӢӨ. м җмҲҳлҘј лӮҙлҰјм°ЁмҲңмңјлЎң м •л ¬н•ң л’Ө мғҒмң„ Kк°ңмқҳ мқёлҚұмҠӨлҘј м„ нғқн•ҳкі , к·ё м•Ҳм—җ нҸ¬н•Ёлҗң мӢӨм ң м–‘м„ұ(мӮ¬кё°) кұҙмҲҳмқҳ н•©мқ„ TPлЎң к°„мЈјн•ҳм—¬ Precision@KмҷҖ Recall@KлҘј кі„мӮ°н•ңлӢӨ. Precision@KлҠ” мғҒмң„ Kкұҙ мӨ‘ мӢӨм ң мӮ¬кё° л№„мңЁмқҙл©°, Recall@KлҠ” м „мІҙ мӮ¬кё° мӨ‘ мғҒмң„ Kм—җ нҸ¬н•Ёлҗң л№„мңЁмқҙлӢӨ.

recall_at_precision н•ЁмҲҳлҠ” м •л°ҖлҸ„ н•ҳн•ңмқ„ л§ҢмЎұн•ҳлҠ” мЎ°кұҙм—җм„ң к°ҖлҠҘн•ң мөңлҢҖ мһ¬нҳ„мңЁмқ„ кі„мӮ°н•ҳлҠ” н•ЁмҲҳмқҙлӢӨ. precision_recall_curveлЎң м–»мқҖ м •л°ҖлҸ„ л°°м—ҙкіј мһ¬нҳ„мңЁ л°°м—ҙмқ„ мӮ¬мҡ©н•ҳм—¬ precisionмқҙ target_precision мқҙмғҒмқё кө¬к°„мқ„ м°ҫкі , к·ё кө¬к°„м—җм„ң мһ¬нҳ„мңЁмқҳ мөңлҢ“к°’мқ„ л°ҳнҷҳн•ңлӢӨ. н•ҙлӢ№ мЎ°кұҙмқ„ л§ҢмЎұн•ҳлҠ” кө¬к°„мқҙ м—Ҷмңјл©ҙ к°’мқҙ мЎҙмһ¬н•ҳм§Җ м•ҠмңјлҜҖлЎң NaNмқ„ л°ҳнҷҳн•ҳлҸ„лЎқ кө¬м„ұлҗҳм–ҙ мһҲлӢӨ. мқҙ н•ЁмҲҳлҠ” нҳ„м—…м—җм„ң м •л°ҖлҸ„ лӘ©н‘ңлҘј лЁјм Җ кі м •н•ң л’Ө нҡҢмҲҳмңЁмқ„ мөңлҢҖнҷ”н•ҳлҠ” м •мұ…мқ„ мҲҳм№ҳлЎң мҡ”м•Ҫн•  л•Ң мң мҡ©н•ҳлӢӨ.

eval_scores н•ЁмҲҳлҠ” н•ҳлӮҳмқҳ лӘЁлҚё м җмҲҳм—җ лҢҖн•ҙ м—¬лҹ¬ м§Җн‘ңлҘј н•ң лІҲм—җ кі„мӮ°н•ҙ л”•м…”л„ҲлҰ¬лЎң л°ҳнҷҳн•ҳлҠ” лһҳнҚј н•ЁмҲҳмқҙлӢӨ. мһ…л ҘмңјлЎң лӘЁлҚё мқҙлҰ„, м •лӢө лқјлІЁ, м җмҲҳлҘј л°ӣкі , ROC-AUCмҷҖ PR-AUC(AP)мқ„ кё°ліё м§Җн‘ңлЎң кі„мӮ°н•ңлӢӨ. лҳҗн•ң Recall@PrecвүҘp0 нҳ•нғңмқҳ м§Җн‘ңлҘј м¶”к°Җн•ҳм—¬ м •л°ҖлҸ„ м ңм•Ҫ н•ҳм—җм„ңмқҳ мөңлҢҖ мһ¬нҳ„мңЁмқ„ н•Ёк»ҳ кё°лЎқн•ңлӢӨ. мқҙм–ҙм„ң k_listм—җ нҸ¬н•Ёлҗң к°Ғ Kм—җ лҢҖн•ҙ Precision@KмҷҖ Recall@KлҘј кі„мӮ°н•ҙ кІ°кіј л”•м…”л„ҲлҰ¬м—җ м¶”к°Җн•ңлӢӨ. кІ°кіјм ҒмңјлЎң н•ң лӘЁлҚём—җ лҢҖн•ҙ мһ„кі„к°’мқҙ м—ҶлҠ” мҲңмң„ кё°л°ҳ м§Җн‘ңмҷҖ мҡҙмҳҒ м •мұ… кё°л°ҳ м§Җн‘ңлҘј лҸҷмӢңм—җ нҷ•ліҙн•ҳлҠ” кө¬м„ұмқҙлӢӨ.

import numpy as np
import pandas as pd

from sklearn.metrics import (
    roc_auc_score, average_precision_score,
    precision_recall_curve
)

def precision_recall_at_k(y_true, score, k):
    """
    мғҒмң„ Kк°ңлҘј м–‘м„ұ(мӮ¬кё°)мңјлЎң ліҙлӮҙлҠ” м •мұ…м—җм„ң
    Precision@K, Recall@K кі„мӮ°
    """
    y_true = np.asarray(y_true).astype(int)
    score = np.asarray(score)

    k = int(min(k, len(score)))
    idx = np.argsort(-score)[:k]
    tp = y_true[idx].sum()
    prec_k = tp / max(k, 1)
    rec_k  = tp / max(y_true.sum(), 1)
    return prec_k, rec_k

def recall_at_precision(y_true, score, target_precision=0.80):
    """
    Precision >= target_precision мқ„ л§ҢмЎұн•ҳлҠ” кө¬к°„ мӨ‘
    Recallмқҳ мөңлҢҖк°’(к°ҖлҠҘн•ҳл©ҙ) л°ҳнҷҳ
    """
    y_true = np.asarray(y_true).astype(int)
    score = np.asarray(score)

    prec, rec, thr = precision_recall_curve(y_true, score)
    mask = prec >= target_precision
    if mask.any():
        return float(np.max(rec[mask]))
    return np.nan

def eval_scores(name, y_true, score, k_list=(100, 200, 500), target_precision=0.80):
    """
    кіөнҶө м§Җн‘ң:
    - ROC-AUC
    - PR-AUC(AP)
    - Precision@K / Recall@K (Top-K мҡҙмҳҒ м •мұ…)
    - Recall @ Precision>=p0 (м •л°ҖлҸ„ м ңм•Ҫ н•ҳм—җм„ң нҡҢмҲҳмңЁ)
    """
    y_true = np.asarray(y_true).astype(int)
    score = np.asarray(score)

    out = {
        "model": name,
        "ROC-AUC": roc_auc_score(y_true, score),
        "PR-AUC(AP)": average_precision_score(y_true, score),
        f"Recall@PrecвүҘ{target_precision:.2f}": recall_at_precision(y_true, score, target_precision)
    }

    for k in k_list:
        pk, rk = precision_recall_at_k(y_true, score, k)
        out[f"P@{k}"] = pk
        out[f"R@{k}"] = rk

    return out

мқҙ мҪ”л“ңлҠ” м„ё к°Җм§Җ лӘЁлҚёмқҙ л§Ңл“  м җмҲҳ(score)лҘј мһ…л ҘмңјлЎң л°ӣм•„, мҡҙмҳҒм—җм„ң мһҗмЈј м“°лҠ” м„ё к°Җм§Җ мһ„кі„к°’ м •мұ…мқ„ лҸҷмқјн•ң нҳ•мӢқмңјлЎң нҸүк°Җн•ҳкі  к·ё кІ°кіјлҘј df_thrлқјлҠ” н‘ңлЎң м •лҰ¬н•ҳлҠ” мҪ”л“ңмқҙлӢӨ. м—¬кё°м„ң scoreлҠ” к°’мқҙ нҒҙмҲҳлЎқ мӮ¬кё°(м–‘м„ұ)мқј к°ҖлҠҘм„ұмқҙ нҒ° м җмҲҳлқјкі  к°Җм •н•ҳл©°, л¶„лҘҳлӘЁлҚёмқҖ мҳҲмёЎнҷ•лҘ  pлҘј, мҳӨнҶ мқёмҪ”лҚ”лҠ” мһ¬кө¬м„ұмҳӨм°ЁлҘј к·ёлҢҖлЎң scoreлЎң мӮ¬мҡ©н•ңлӢӨ.

import numpy as np
import pandas as pd
from sklearn.metrics import confusion_matrix, precision_recall_curve

# -----------------------------
# кіөнҶө нҸүк°Җ н•ЁмҲҳ
# -----------------------------
def eval_threshold(y_true, score, thr, positive_if="ge"):
    """
    positive_if:
      - "ge": score >= thr мқҙл©ҙ м–‘м„ұ
      - "le": score <= thr мқҙл©ҙ м–‘м„ұ (кұ°мқҳ м•Ҳ м”Җ)
    """
    y_true = np.asarray(y_true).astype(int)
    score = np.asarray(score)

    if positive_if == "ge":
        y_pred = (score >= thr).astype(int)
    else:
        y_pred = (score <= thr).astype(int)

    tn, fp, fn, tp = confusion_matrix(y_true, y_pred).ravel()
    precision = tp / (tp + fp + 1e-12)
    recall    = tp / (tp + fn + 1e-12)
    fpr       = fp / (fp + tn + 1e-12)
    tpr       = recall
    return {
        "thr": float(thr),
        "TP": int(tp), "FP": int(fp), "FN": int(fn), "TN": int(tn),
        "precision": float(precision),
        "recall": float(recall),
        "fpr": float(fpr),
        "tpr": float(tpr),
        "pred_pos": int(tp + fp)
    }

def find_topk_threshold(score, k):
    score = np.asarray(score)
    k = int(min(k, len(score)))
    idx = np.argsort(-score)          # descending
    thr = score[idx[k-1]]             # KлІҲм§ё м җмҲҳ
    return float(thr)

def find_cost_threshold(y_true, score, cFN=20.0, cFP=1.0, grid_q=400):
    """
    л¶„мң„мҲҳ к·ёлҰ¬л“ңм—җм„ң л№„мҡ© мөңмҶҢ мһ„кі„к°’ нғҗмғү
    """
    y_true = np.asarray(y_true).astype(int)
    score = np.asarray(score)

    cand = np.unique(np.quantile(score, np.linspace(0, 1, grid_q)))
    best = None
    for thr in cand:
        d = eval_threshold(y_true, score, thr, positive_if="ge")
        cost = cFN * d["FN"] + cFP * d["FP"]
        d["cost"] = float(cost)
        if (best is None) or (d["cost"] < best["cost"]):
            best = d
    return best

def find_precision_constrained_threshold(y_true, score, p0=0.80):
    """
    PR curve кё°л°ҳмңјлЎң Precision >= p0 лҘј л§ҢмЎұн•ҳлҠ” м§Җм җ мӨ‘ Recall мөңлҢҖк°Җ лҗҳлҠ” threshold м„ нғқ
    """
    y_true = np.asarray(y_true).astype(int)
    score = np.asarray(score)

    prec, rec, thr = precision_recall_curve(y_true, score)
    # thr кёёмқҙ = len(prec)-1 мқҙлҜҖлЎң м •л ¬ л§һм¶”кё°
    thr = np.concatenate([thr, [1.0]])

    mask = prec >= p0
    if not mask.any():
        return None  # л¶Ҳк°ҖлҠҘ
    # recall мөңлҢҖ, лҸҷлҘ мқҙл©ҙ precision нҒ° кІғ м„ нғқ
    idx = np.argmax(np.where(mask, rec, -1))
    chosen_thr = thr[idx]
    d = eval_threshold(y_true, score, chosen_thr, positive_if="ge")
    d["p0"] = float(p0)
    return d

def eval_threshold(y_true, score, thr, positive_if="ge"):
    y_true = np.asarray(y_true).astype(int)
    score  = np.asarray(score)

    if positive_if == "ge":
        y_pred = (score >= thr).astype(int)
    else:
        y_pred = (score <= thr).astype(int)

    tn, fp, fn, tp = confusion_matrix(y_true, y_pred).ravel()
    precision = tp / (tp + fp + 1e-12)
    recall    = tp / (tp + fn + 1e-12)
    fpr       = fp / (fp + tn + 1e-12)

    return {
        "thr": float(thr),
        "TP": int(tp), "FP": int(fp), "FN": int(fn), "TN": int(tn),
        "precision": float(precision),
        "recall": float(recall),
        "fpr": float(fpr),
        "pred_pos": int(tp + fp)
    }

def find_topk_threshold(score, k):
    score = np.asarray(score)
    k = int(min(k, len(score)))
    idx = np.argsort(-score)
    return float(score[idx[k-1]])

def find_cost_threshold(y_true, score, cFN=20.0, cFP=1.0, grid_q=500):
    y_true = np.asarray(y_true).astype(int)
    score  = np.asarray(score)
    cand = np.unique(np.quantile(score, np.linspace(0, 1, grid_q)))

    best = None
    for thr in cand:
        d = eval_threshold(y_true, score, thr, positive_if="ge")
        d["cost"] = float(cFN * d["FN"] + cFP * d["FP"])
        if (best is None) or (d["cost"] < best["cost"]):
            best = d
    return best

def find_precision_constrained_threshold(y_true, score, p0=0.80):
    y_true = np.asarray(y_true).astype(int)
    score  = np.asarray(score)

    prec, rec, thr = precision_recall_curve(y_true, score)
    thr = np.concatenate([thr, [1.0]])  # кёёмқҙ л§һм¶Ө

    mask = prec >= p0
    if not mask.any():
        return None

    idx = np.argmax(np.where(mask, rec, -1))
    chosen_thr = float(thr[idx])
    d = eval_threshold(y_true, score, chosen_thr, positive_if="ge")
    d["p0"] = float(p0)
    return d
